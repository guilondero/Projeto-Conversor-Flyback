# Projeto-Conversor-Flyback

Projeto referente ao trabalho 1 da cadeira de Eletrônica de potência do curso de Engenharia de Controle e Automação - UFSM. O projeto é referente aos cálculos do filtro capacitivo do retificador de meia onda completa e aos cálculos do indutor e capacitor do conversor Flyback.
